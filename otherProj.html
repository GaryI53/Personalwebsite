Other projects

3D printed Keyboard

For 3D printing fans, this was printed on a Ender 3 V2 with BL Touch in PETG using the Sick86 template. It was made in four pieces due to the lack of space on my print bed. I installed new yellow gateron switches which were lubes and had gaskets installed. I then took the perilous task of soldering all the switches and diodes to the Teensy board to create a matrix to which I could take inputs. I then created a Python script to create the required key output with each button press.

Hacksussex 2025 main prize winning project - Ball detection with YoloV11 with reframe

HackSussex hackathon 25 main prize winning project taking regular wide angle action camera footage and converting this in to broadcast style footage. Custom trained model using a mix of custom and Coco datasets to create object detection for the ball allowing us to determine where to reframe the image, providing a smoother more realistic transition to real-time mechanical/ml solutions. (Solo project)

Custom 3D printed mouse using Lidar scanning for true ergonomics

3D printed mouse using the Bambu labs hardware kit alongside a custom shell. The custom shell was designed on blender by creating a boolean of my lidar scanned hand. The custom shell was then made to fit with the wireless bambu kit.

Lithium battery/IPS screen Gameboy

I'm very passionate about 90's gaming, especially Nintendo. Growing up with a Gameboy color I really wanted to mod one to modernise it and overcome it's shortcomings. So I used the FunnyPlaying mods to add a much improved full colour IPS display with touch sensor. I then used a dremel to cut out the rear compartment so I could solder in a rechargeable lithium battery. Goodbye double A's.
